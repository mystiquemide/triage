# Security Policy

## Reporting a Vulnerability

Please do not report sensitive vulnerabilities in public GitHub issues.

Email or privately contact the project maintainer with:

- A clear summary of the issue
- Impact and affected components
- Reproduction steps or proof of concept
- Suggested remediation, if available

## Scope

This repository contains:

- Vue/Vite frontend code
- GenLayer intelligent contract code
- Direct-mode contract tests and deployment helpers

Do not submit real private keys, seed phrases, wallet secrets, or production credentials in reports.

## Current Network Status

The app targets GenLayer StudioNet. Patched contract redeploys may remain pending while StudioNet validators/RPC are delayed. Always verify the configured contract address before relying on a deployment.
